SQL — запити та агрегації:
SELECT, JOIN, GROUP BY та підзапити

SQL — мова аналітика. Вміння писати запити дозволяє тобі самостійно перевіряти дані, будувати звіти і знаходити аномалії без розробника.

🔑 Основні SQL операції

SELECT з фільтрацією:

-- Знайти активних BA у системі SELECT id, name, email, role FROM users WHERE role = 'BA' AND active = true AND created_at > '2024-01-01' ORDER BY name ASC LIMIT 50;

Агрегації:

-- Статистика замовлень по статусах SELECT status, COUNT(*) AS total, SUM(amount) AS revenue, AVG(amount) AS avg_order FROM orders GROUP BY status HAVING COUNT(*) > 10 ORDER BY revenue DESC;
🔗 JOIN — з'єднання таблиць

INNER JOIN приклад:

-- Замовлення з клієнтами та товарами SELECT c.name AS customer, o.OrderDate, p.ProductName, od.Quantity, od.Price FROM Orders o JOIN Customers c ON o.CustomerID = c.CustomerID JOIN OrderDetails od ON o.OrderID = od.OrderID JOIN Products p ON od.ProductID = p.ProductID ORDER BY o.OrderDate;

LEFT JOIN — клієнти без замовлень:

-- Хто жодного разу не замовляв? SELECT c.CustomerID, c.Name, c.Email FROM Customers c LEFT JOIN Orders o ON c.CustomerID = o.CustomerID WHERE o.OrderID IS NULL; -- Якщо o.OrderID IS NULL значить -- для цього клієнта немає замовлень
🧩 Підзапити та практичні приклади

Subquery — підзапит:

-- Клієнти що витратили більше середнього SELECT c.Name, SUM(o.TotalAmount) AS total FROM Customers c JOIN Orders o ON c.CustomerID = o.CustomerID GROUP BY c.Name HAVING SUM(o.TotalAmount) > ( SELECT AVG(TotalAmount) FROM Orders ) ORDER BY total DESC;

Топ-3 клієнти:

-- Топ-3 клієнти за сумою замовлень SELECT c.Name, COUNT(o.OrderID) AS orders_count, SUM(o.TotalAmount) AS total_spent FROM Customers c JOIN Orders o ON c.CustomerID = o.CustomerID GROUP BY c.Name ORDER BY total_spent DESC LIMIT 3;
🎯 BA і SQL: Вимога "звіт: всі замовлення з іменами клієнтів за цей місяць" = JOIN + WHERE по даті. Знаючи SQL ти пишеш точніші вимоги до звітів і можеш самостійно перевіряти дані без розробника.
📊 Практичні завдання
  • Напиши SELECT що повертає всіх клієнтів з містом Kyiv відсортованих за ім'ям
  • Знайди топ-5 продуктів за загальною кількістю продажів
  • Порахуй середню суму замовлення по кожному клієнту (GROUP BY + AVG)
  • Знайди клієнтів що зробили більше 2 замовлень за місяць (HAVING)
  • Виведи продукти без жодного замовлення (LEFT JOIN + IS NULL)
